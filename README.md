# JHU Data Scientist's Toolbox

This repository was created for the **Johns Hopkins University Data Scientist's Toolbox** course on Coursera.

The repository documents the basic course setup for working with **R**, **RStudio**, **Git**, **GitHub**, and **R Markdown**.

## Course Project

The course project, **Assemble Your Toolbox**, focuses on confirming that the basic data science tools introduced in the course are installed, configured, and working together.

This repository supports that project by showing a working local RStudio project connected to GitHub through Git version control.

---

## Tools Demonstrated

- R
- RStudio
- Git
- GitHub
- R Markdown

---

## Notes

This repository is intentionally simple. It is not a full data analysis project. It is a setup and workflow repository for the first course in the Johns Hopkins Data Science Specialization.

The main assignment evidence is documented in the R Markdown file and its rendered output.
